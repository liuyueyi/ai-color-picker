## ai-color-picker

基于AI生成的颜色提取项目

### 开发工具栈

- [V0.dev](https://v0.dev/)
- [Trae](https://www.trae.ai/)


- h5 打包之后，托管到 Cloudflare Pages，对应的访问地址:
    - [https://colorpicker-bpx.pages.dev/#/](https://colorpicker-bpx.pages.dev/#/)


### 能力

1. 上传图片，提取颜色
2. 记录提取的颜色信息
3. 收藏颜色
